# Student Course & Performance Management System (SQL)

## Description
This project implements a normalized SQL database to manage students, courses, enrollments, and academic performance.

## Features
- Student and course management
- Many-to-many relationship using enrollments
- Marks and grade calculation using SQL views
- Analytical queries for performance evaluation

## Technologies Used
- MySQL
- SQL
- VS Code

## Files
- schema.sql – Database schema
- sample_data.sql – Sample records
- queries.sql – Views and analytical queries

## How to Run
1. Create database using schema.sql
2. Insert data using sample_data.sql
3. Execute queries.sql to view results
